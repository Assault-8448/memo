## 重要そうなメモ

- ffmpegは高解像度になればなるほどGPUゴリゴリ使ってくれる。  
  1920x1080ではVideo Encodeを10%を使ってくれない。4Kあるいはそれ以上になるともっと使ってくれる・・・のか？（未検証）

- avsファイルはAviutlに食わせる際コーデックを色々入れないと「could not read...」エラーを吐く。
  「LAVFilters」や「ffdshow」を入れると動いてくれる。
